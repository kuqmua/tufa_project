# Naming conventions

## Suffixes
- `_ts` TokenStream
- `_sc` snake case string
- `_ucc`upper camel case string
- `_dq` double-quoted string
- `_h` handle

## Domain abbreviations
- `pk` primary key
- `fi` field (in iteration context)
- `ft` field type
- `cr` create
- `rd` read
- `upd` update
- `del` delete
- `sel` select
- `qb` query bind
- `qp` query part
- `nn` non-null
- `nl` nullable
- `tt` table_type
- `pgn` pagination
- `acc` accumulator
- `vrt` variant
- `ident` syn::Ident
- `inn` inner
- `dflt`default
- `dcl` declaration
- `init`initialization
- `incr`increment
- `pgn` pagination
